Bienvenido a PathPic.

Esta utilidad online esta pensada para aquellos desarrolladores que quieran que los paths que coloquen en una página web tengan una mejor presentacion e incluso con el estilo de otro sistema operativo diferente al que tengan disponible. La dirección es : http://pathpic.pccore.net/index.php

Formato de rutas

Las rutas pueden ser escritas en formato windows o linux. Pueden comenzar o no con un forwardslash /var/usr/ ó var/usr Pueden tener una estructura de proyecto, como : /project/src /project/public /project/library /project/test

O en formato Windows : \project\src \project\public \project\library \project\test

El formato de rutas debera coincidir con la opcion Formato de Paths para evitar errores.

Seleccionar un estilo de árbol

Los estilos disponibles son :

  • Windows XP
  • Windows 7
  • Ubuntu
  • Netbeans
  • Eclipse
  • Mac

Indicar Archivos y Highlights

Para indicar que un elemento (path) corresponde a una archivo, se debera colocar en el casillero su número de fila comenzando en cero (0) en el casillero Archivos separado por comas y si no existiera ningun archivo en la estructura de paths, dejar este casillero vacío. Si el path del archivo esta en la primera linea colocar en el casillero Archivo el numero cero (0).

De la misma forma se indica si se quiere resaltar un elemento colocaldo su número de fila en el casillero HighLights.
